## ast 的过程
*  1:    词法分析 [ var, a, = , 1 , +, 2] = token
     *    正则表达式
          	*  判断这些token里面的字符串是否是合法的字符表达
          	*  循环token
          	*  判断当前字符串是否是关键字【函数，或者变量声明】
   *  状态机
      		*   1： 处理字符串的状态
      		*   2： 遍历空格
      		*   3： 处理字符串状态
      		*   4： 符号
      		*   5： 处理数字

*  2： 语法分析
*  3： 遍历ast树
*  4： 如果修改了某一个节点， 就重新遍历ast树
*  5： ast => code
