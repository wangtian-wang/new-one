## 声明空间
* 变量声明空间
    *  const let var function class
* 类型声明空间
    *  interface type class
* type 和interface的区别
    * interface
        * interface per {
            name: string
        }
    * type 类型别名: 通过引用，来使用定义好的type对象
        * type per = {
             name: string
        } 采用的是赋值的形式,可以赋值为任何的值
## 类型别名可以以对象属性的形式去使用自身，但是不能直接赋值。
## 当类型别名为接口增加别名的时候不能使用 extend implements
