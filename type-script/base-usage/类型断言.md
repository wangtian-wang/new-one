### Type Assertion : 可以用来手动指定一个值的类型。
### 语法：<类型> 值  ||  值 as 类型
### 举例：
```
 function get (x:number | string) {
     // 写法1
     if((<string>x).length) {
         return (<string>x).length
     }
     // 写法2
      if((x as string).length) {
         return (x as string).length
     }
 }
```
